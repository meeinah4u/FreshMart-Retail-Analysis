# FreshMart-Retail-Analysis

## 📊 Project Overview
This project involved a comprehensive analysis of FreshMart Retail Ghana's sales data. The goal was to transform raw, unstructured data into meaningful business intelligence to improve operational efficiency and decision-making.

## 🛠 Tools Used
* **Microsoft Excel:** Data Cleaning, Sorting, Filtering, and Advanced Conditional Formatting.
* **Data Visualization:** Using Data Bars and Color Scales to identify performance trends.

## 🚀 Key Insights
* **Top Revenue Driver:** Frozen Chicken (Key staple item).
* **Highest Volume Product:** Milk Tin (Large) (Primary driver of customer traffic).
* **Top Performing Region:** Accra (Outperforms all other regions in revenue and sales volume).
* **Inventory Efficiency:** Identified products with low sales velocity (Units Sold < 30) for inventory optimization.
* **Regional Disparity:** Customer demand patterns vary significantly across regions.
